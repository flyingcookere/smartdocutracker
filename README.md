# SmartDocuTracker

SmartDocuTracker is a web-based document tracking system developed to streamline the monitoring and management of official documents within the Faculty of Engineering at Bulacan State University (BulSU). The system enables faculty members and administrators to efficiently track the movement of documents between offices, including the College Dean's Office and faculty departments.

## Overview

Managing physical and administrative documents can be challenging when multiple offices are involved. SmartDocuTracker provides real-time visibility into the location and status of documents, reducing delays, improving accountability, and minimizing the risk of misplaced records.

## Key Features

* Document registration and tracking
* Real-time document status monitoring
* Tracking of document transfers between offices
* Document history and movement logs
* Status updates throughout the document lifecycle
* User-friendly dashboard for quick access to tracking information
* Secure user authentication and role-based access

## Purpose

The system was designed to improve document handling processes within the Faculty of Engineering by providing a centralized platform where users can easily determine:

* The current location of a document
* The current processing status
* The office responsible for the document
* The complete movement history of the document

## Benefits
